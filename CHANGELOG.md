# Changelog

All notable changes to CosyVoice Studio will be documented in this file.

## [2.0.0] - 2026-03-15

### Added
- **批量生成队列** — 每行一条文本，逐条排队生成，进度条实时显示（⏳→🔄→✅/❌）
- **ZIP 打包下载** — 批量生成完成后自动打包为 ZIP 一键下载
- **格式选择器** — WAV（无损）/ MP3（更小）双格式切换
- **语义化文件命名** — 下载文件名格式 `{音色}_{文本前缀}_{序号}.wav`
- **声音库标签系统** — 保存声音时可添加标签，逗号分隔
- **时长预估** — 根据文字数量估算生成时间，显示在 Loading 和批量模式中
- **单条/批量模式切换** — 预设音色 Tab 支持一键切换

### New Files
- `src/batch.js` — 批量生成引擎（队列执行 + ZIP 打包 + 时长估算）
- `src/format.js` — 音频格式转换（WAV→MP3 + 语义命名生成器）

### Dependencies
- Added `jszip` ^3.10.1 — ZIP 打包
- Added `@breezystack/lamejs` ^1.2.7 — MP3 编码

---

## [1.0.0] - 2026-03-14

### Added
- **四模式 Tab 系统** — 声音克隆 / 预设音色 / 跨语言合成 / 指令控制
- **预设音色合成** (SFT) — 7 种内置音色，中/英/日/粤/韩
- **声音克隆** (Zero-Shot) — 上传/录制参考音频 + 文字 → 克隆
- **跨语言合成** (Cross-Lingual) — 参考音色 + 任意语言文本
- **指令控制** (Instruct2) — 自然语言情感/语速指令
- **录音模块** — 浏览器内录音，WAV 编码，15 秒时长保护
- **波形播放器** — Canvas 实时波形可视化
- **历史记录** — IndexedDB 持久化，回听/下载
- **声音库** — 保存/加载/删除克隆的声音 Profile
- **智能模式感知** — 自动检测服务端模型能力，禁用不支持的 Tab
- **动态 Speaker Grid** — 从服务端获取音色列表动态渲染
- **文本模板** — 多语言示例文本一键填充（中/英/日/法）
- **CORS 代理** — Vite proxy `/api` → CosyVoice 服务端
- **暗色主题** — 玻璃拟态设计，渐变色 UI
